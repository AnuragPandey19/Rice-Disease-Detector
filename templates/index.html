      About the Project

      Transfer Learning with ResNet18

      This project uses state-of-the-art deep learning techniques to identify rice leaf diseases. Built on the ResNet18 architecture pre-trained on ImageNet, the model has been fine-tuned specifically for rice disease classification.

      Detected Diseases

      • Bacterial Leaf Blight: Serious bacterial infection causing wilting
      • Brown Spot: Fungal disease with characteristic brown lesions
      • Leaf Smut: Fungal infection producing black spore masses

      Technology Stack

      PyTorch ResNet18 Flask Transfer Learning Computer Vision
